🎯 项目简介

VFT_SMF（Virtual Flight Test Simulation and Modeling Framework）是一个基于数字孪生和事件驱动架构的航空仿真平台，专为虚拟试飞和飞行仿真研究设计。

本项目采用现代化的C++17标准，结合多线程技术和实时数据处理，为航空仿真领域提供高性能、可扩展的解决方案。

🏗️ 数字孪生架构

三层模型设计：数据层、模型层、服务层，实时数据同步，完整的生命周期管理。

⚡ 事件驱动设计

异步事件处理，多线程架构，毫秒级事件处理能力，充分利用多核处理器性能。

🤖 五代理建模系统

飞行员、飞机系统、环境、ATC、飞行动力学五大代理，精确模拟复杂航空系统。

📊 数据管理

实时数据记录，可视化分析，配置驱动，JSON配置文件管理，高性能数据采集。

🚀 快速开始

# 克隆仓库
git clone https://github.com/GuilinWuGit/VFT_SMF.git

# 进入项目目录
cd VFT_SMF

# 编译项目
cd ScenarioExamples/B737_Taxi
./build.bat

# 运行仿真
./EventDrivenSimulation_NewArchitecture.exe
